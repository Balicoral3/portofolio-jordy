# Jordy Samahati – Frontend Engineer Portfolio

Live website: [jordysamahati.com](https://jordysamahati.com)

## Tech Stack
- HTML5
- CSS3 (custom properties, flexbox, animations)
- Vanilla JavaScript (Intersection Observer, smooth scroll)
- Google Fonts (Inter)

## Features
- Dark mode aesthetic with subtle gradient background
- Sticky navigation with backdrop blur
- Entrance animations on scroll (fade-up)
- Responsive design (mobile-friendly)
- Structured data (JSON-LD) for SEO
- Security headers via `.htaccess`

## Why This Portfolio?
I built this single‑page portfolio from scratch without any frameworks to demonstrate my understanding of core web fundamentals. The design is inspired by modern startup aesthetics (Stripe, Linear), and the content highlights my unique journey from e‑commerce operations to frontend engineering.
